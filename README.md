# PHBPlugin
 Um plug-in completo para o Arcturus Emulator e em constante atualizações.
 
# Compra
- Pagamento único.
- Atualizações ilimitadas.
- Suporte para instalação.

# Comandos originais modificados
```
:about - Comando about modificado, traduzido e com informações de pico de usuários e quartos.
:empty - Comando de empty modificado, onde não apaga os raros ltd e gera suas moedas e diamantes antes de remover.
:userinfo - Comando de user info modificado, com informações dos pontos do premiar.
:invisible - Comando invisible modificado, onde você continuará invisível caso mude de quarto, só desativando através do comando.
:sit - Comando sit modificado com cooldown.
```
# Comandos de ferramentas
```
:about - Comando about modificado, traduzido e com informações de pico de usuários e quartos.
:empty - Comando de empty modificado, onde não apaga os raros ltd e gera suas moedas e diamantes antes de remover.
:userinfo - Comando de user info modificado, com informações dos pontos do premiar.
:invisible - Comando invisible modificado, onde você continuará invisível caso mude de quarto, só desativando através do comando.
:sit - Comando sit modificado com cooldown.
:buyroom - Comando de comprar quarto do Rosie, porém com verificação de tradeban.
:sellroom - Comando de vender quarto do Rosie, porém com verificação de tradeban.
:furnidata - Exibe informações do item.
:editaritem - Edita parâmetros da tabela items_base.
:ondeta - Exibe em que página do catálogo o item se encontra.
:dj - Informa através de uma notificação do tipo bubble a programação da rádio atual.
:aq - Abre o quarto atual.
:fq - Fecha o quarto atual.
:loteriacomprar - Compre um bilhete da loteria com esse comando.
:loteria - Liste, conte ou premie os participantes da loteria com esse comando.
:eventosoff - Desativa a capacidade de receber novos alertas de evento.
:eha - Alerta de evento através de websockets, que abre e fecha o quarto automaticamente.
:prefixo - Muda a tag que aparece antes do seu nome.
:prefixocor - Muda a cor da sua tag.
:nomecor - Muda a cor do seu nome.
:textocor - Muda a cor do texto.
:fakes - Mostra a lista de possíveis fakes, baseado na quantidade de contas logadas no mesmo endereço de IP.
:tp - Teletransporta o usuário especificado para a sua frente.
:tproom - Teletransporta todos os usuários do quarto para a sua frente.
:roompublic - Transforma o quarto atual em um quarto público.
:roomvideo - Exibe um vídeo no quarto, através de websockets.
:hotelvideo - Exibe um vídeo no hotel inteiro, através de websockets.
:roomimage - Exibe uma imagem no quarto, através de websockets.
:hotelimage - Exibe uma imagem no hotel inteiro, através de websockets.
:premiar - Premia um usuário com créditos, dá um emblema e adiciona um novo ponto.
:give - Dá moedas a um usuário.
:givepoints - Dá pontos a um usuário.
:coloradd - Através do websockets, abre uma imagem que auxilia pessoas com daltonismo.
:cataltd - Atualiza o catálogo e envia um alerta alertando de um novo raro disponível.
:bemvindo - Menciona um usuário desejando boas-vindas ao hotel.
:mudardono - Muda o dono de um quarto.
:pay - Paga um usuário.
:closedices - Fecha as dices ao seu redor.
:rolldice - Roda as dices ao seu redor.
:ra - Manda alerta para um determinado rank.
:staffson - Mostra uma lista com staffs online.
:userson - Mostra uma lista com os usuários online.
:inv - Vê o inventário do usuário mencionado.
:flaguser - Ativa ou desativa a troca de nome do usuário mencionado.
:builder - Comando tele que é ativado somente se o usuário for dono do quarto.
:blockcmd - Bloqueia um comando na sala. (há uma permissão para override).
:nitrar - Comando que habilita determinados ranks pré configurados a entrar automaticamente no quarto após o eventalert ser enviado.
:youtube [nome do video] - Pega o vídeo com o titulo correspondente, por meio da API v3 do Youtube. (configure sua api_key em emulator_settings).
```

# Interações
```
InteractionArcas - Sistema de arcas, onde você pode definir ilimitados prêmios para a mesma e será dado um prêmio aleatório.
InteractionCorNome - Responsável por permitir compras de cores de nome pela loja.
InteractionCorPrefixo - Responsável por permitir compras de cores de prefixo pela loja.
InteractionLoteria - Responsável por permitir compra de bilhete de loteria pela loja.
InteractionPrefixo - Responsável por permitir compras de prefixo pela loja.
InteractionBotemPlanet - Planeta botem. (Botem é como se fosse o totem, porém com efeitos diferentes e configuráveis no emulator_settings).
InteractionBotemHead - Corpo do botem.
InteractionBotemLegs - Pés do botem.
```

# Funções extras
```
Sistema de prefixo, cor de nome, cor de prefixo e cor de texto personalizados.
Sistema de loteria através do catálogo ou por comando.
Cooldown de comandos com tempo em segundos configuráveis através da tabela emulator_settings.
Limite da feira livre configurável.
Correção do bug da roleta (foi adicionado um cooldown)
Nova permissão para staffs removerem mobis que não são deles em outros quartos. (acc_removermobis)
Alertas de compra de ltd configuráveis pela permissão "acc_hotel.alert.ltd.buy".
Nova tabela de logs de comandos staffs, que salva o quarto onde foi executado (phbcommandlogs).
Sistema de configurar quantidade de respeitos através do rank, em permissions (respects,respectspet).
Sistema de pico de usuários e quarto, onde é salvado na tabela "graficos_phbplugin" sendo possível criar um gráfico em seu HK.
Sistema de websockets aprimorado, onde verifica se usuário recebeu a mensagem enviada, e se não, faz 10 tentativas em um intervalo de 300ms para garantir que seja entregue.
Exibição da quantidade de respeitos recebidos no perfil do usuário.
Bloqueio na utilização da feira livre para quem está com tradeban.
Sistema anti-flop de comandos, caso o comando digitado não exista, ele retorna um sussurro e não mostra o comando que você digitou. (configurável através do emulator_settings).
Sistema de log de comandos staffs com informações do quarto.
Sistema de logs de feira livre.
```
